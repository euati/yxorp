Для приложения ShadowsocksR

Перед настройкой в первую очередь вы должны быть зарегистрированы и авторизованы.

1. Скачиваем ShadowsocksR (SSR).

2. Копируем URL подписку в личном кабинете

3. Открываем ShadowsocksR. В правом нижнем углу нажимаем правой кнопкой мыши на иконку самолетика.
Идём в “Node Subscribe” → “Subscribe Setting”. Вставить ранее скопированную ссылку URL, включить галочку “Auto update” и нажать “OK”.
Снова идём “Node Subscribe” → “Update subscribe(bypass proxy)”. Если видите уведомление “Success ..”, все сервера загрузились.

4. Идём в “Mode” ставим Global (Если хотите чтобы весь трафик проходил через прокси) или PAC (Если только заблокированные сайты).

5. Сервер выбираем “Servers” → “XYZ” → “Нужный сервер”







Load balance-программа автоматически выбирает наиболее подходящий сервер.



Для приложения ShadowsocksR

Перед настройкой в первую очередь вы должны быть зарегистрированы и авторизованы.

1. Скачиваем ShadowsocksR> и Устанавливаем.

2. На странице Personal account копируем ссылку Subscription URL.
открываем установленное приложение, на левом верхнем углу нажимаем на логотип shadowsocks R,
нажимаем на круглую кнопку, выбираем Add/Update SSR Subscription, свайпом влево удаляем Free SSR public.
Нажимаем Add URL вставляем ранее скопированную ссылку >> "ok" >> включить "Auto Update", нажимаем "Сохранить и обновить".

Удаляем сервер по умолчанию (8388) свайпом влево.
Включаем IPv6 Route и UDP Forwarding.

3. Выбираем сервер, нажимаем на круглую
кнопку самолетика. "Подключаемся", проверяем











Для приложения ShadowsocksX-NG.

Перед настройкой в первую очередь вы должны быть зарегистрированы и авторизованы.

1. Скачиваем ShadowsocksX-NG-R8.

2. Скачиваем конфигурацию.

3. Открываем скачанный архив, программу перетаскиваем в приложения




4. Запускаем программу, после, в правом верхнем углу видим значок самолетика, идем в Servers >> Import Bunch Json File...,
в открывшемся окне находим нашу скачанную конфигурацию (ss.json)





5.Идем в Servers выбираем сервер который вам нужен, Shadowsocks: On- означает что вы подключены

Пробуем зайти на youtube, facebook, twitter и т.д



Для приложения Potatso Lite

Перед настройкой в первую очередь вы должны быть зарегистрированы и авторизованы.
1. В AppStore ищем Potatso Lite и устанавливаем.
2. На странице Personal account копируем ссылку Subscription URL.
3 . Открыть Potatso Lite выбрать "Get Started" >> "Add now" >> "Subscription" >> написать "Name", включить "Auto update" и вставить ранее скопированную ссылку "Subscription URL". На прав верхнем углу нажать на галочку.
Нажать "Start" и разрешить доступ если смартфон запросит.
Важно! при работе с ВПН надо отключать GPS локацию, так как такие приложения как инстаграм, вк и тд. передают ваше реальное местоположение.






В данном случае использовался Ubuntu 16.04 LTS


Перед настройкой в первую очередь вы должны быть зарегистрированы и авторизованы.
1.Установка В менеджере приложений устанавливаем Shadowsocks-qt5
либо через терминал
pip install shadowsocks
apt-get install shadowsocks-libev
cpan Net::Shadowsocks
2. Добавляем сервера. На странице All servers открываем конфигурацию сервера, копируем ссылку import (SS applications).
3. Открывем приложение. Идем в Connection >> Add >> URL вставляем ссылку жмем OK, пишем имя сервера для удобства.
4. Настраиваем браузер. Открываем настройки браузера (в моем случае Firefox)
идем в настройки прокси, выбираем "Ручная настройка прокси", в "Узел SOCKS" вводим 127.0.0.1 Порт 1080, выбираем SOCKS 5.
Ниже ставим галочку Отправлять DNS-запросы через прокси при использовании SOCKS 5.
Нажимаем OK, проверяем соединение.
Внимание! Если хотите чтобы браузер работал без прокси, надо снова возвращать настройки по умолчанию.
